# @minions-dev-prompts/cli

CLI for Minions Dev-prompts — Reusable prompt templates, prompt chains, and prompt versioning for agent-tasks

## Installation

```bash
npm install -g @minions-dev-prompts/cli
```

## Usage

```bash
dev-prompts --help
```
