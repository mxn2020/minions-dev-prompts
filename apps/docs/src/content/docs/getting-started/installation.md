---
title: Installation
description: Install Minions Dev-prompts packages
---

## TypeScript / Node.js

```bash
npm install @minions-dev-prompts/sdk minions-sdk
```

## Python

```bash
pip install minions-dev-prompts
```

## CLI

```bash
npm install -g @minions-dev-prompts/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
