# Minions Dev-prompts

Reusable prompt templates, prompt chains, and prompt versioning for agent-tasks

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-dev-prompts/sdk`) — Core library for Node.js
- **Python SDK** (`minions-dev-prompts`) — Python bindings
- **CLI Tool** (`@minions-dev-prompts/cli`) — Command-line interface
- **Web App** — Interactive playground at [dev-prompts.minions.wtf](https://dev-prompts.minions.wtf)
